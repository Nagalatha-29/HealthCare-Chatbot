# Healthcare Advice Chatbot

This project is a simple AI-based Healthcare Advice Chatbot.  
Users can ask health-related questions and get general advice and guidelines.

The chatbot uses an AI model to understand the question and give helpful suggestions.

## Features

- Ask healthcare questions
- Get simple health advice
- Supports topics like diabetes, blood pressure, and mental health
- Easy web interface using Streamlit
- Shows previous conversation history

## Technologies Used

- Python
- Streamlit
- LangChain
- LangGraph
- Groq LLM

## How It Works

1. The user asks a healthcare question.
2. The system finds related health guidelines.
3. The AI model analyzes the question.
4. The chatbot gives a helpful response.
